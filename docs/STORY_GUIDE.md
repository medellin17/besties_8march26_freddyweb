# Story Guide

## Общая структура
Каждая версия (Полина / Маша) проходится за **5–10 минут**.
~15–20 сцен, 3–4 точки ветвления, 2–3 концовки.

## Формат сцены (TypeScript)
```ts
interface Scene {
  id: string;
  background: 'office' | 'hallway' | 'stage' | 'party_room' | 'title';
  characters: { name: string; expression: string; position: 'left' | 'center' | 'right' }[];
  dialog: { speaker: string | null; text: string; memeImage?: string }[];
  choices?: { text: string; nextId: string }[];
  nextId?: string; // авто-переход если нет choices
  effect?: 'shake' | 'flash' | 'static' | 'blackout';
  sound?: 'phone_ring' | 'footsteps' | 'jumpscare' | 'ambient';
  isEnding?: boolean;
}
```

## Структура сюжета (обе версии)

### Акт 1 — Прибытие
1. **Титульный экран** → Посвящение: "Для [Имя], с 8 марта! От Глеба 🐤"
2. **Звонок Phone Guy (Глеб):** Приветствие, кринжовые инструкции, абсурдные советы
3. **Первый взгляд на офис:** Описание мониторов, вентиляции, атмосферы

### Акт 2 — Встреча с напарником
4. **Появление напарника** (Бонни для Полины / Фредди для Маши)
5. **Диалог:** Знакомство, шутки, отсылки к Brawl Stars и внутренние мемы
6. **Выбор #1:** Куда пойти — проверить камеры или осмотреть коридор?

### Акт 3 — Инцидент
7. **Свет моргает.** Звуки шагов (Фокси).
8. **Выбор #2:** Закрыть дверь или посмотреть в коридор?
   - Закрыть → Фокси стучит. Напарник шутит.
   - Посмотреть → *Фокси-скример (комедийный)*. Фокси: "ААААА... дай зарядку от айфона"
9. **Мем-момент:** Показ одного из мемов в контексте диалога

### Акт 4 — Кульминация
10. **Встреча всех аниматроников:** Все собираются вместе
11. **Выбор #3:** Финальная реакция — влияет на концовку
12. **Появление Чики (Глеб):** Раскрытие, что Phone Guy = Чика

### Акт 5 — Финал
13. **Поздравление:** Все аниматроники поздравляют с 8 марта
14. **Показ фото 23 февраля** (лица на аниматрониках) как "корпоративное фото"
15. **Финальный текст от Глеба** (искренний, короткий)

## Концовки
- **Хорошая:** Все дружат, пиццерия устраивает вечеринку в честь 8 марта
- **Секретная:** Фокси становится другом и катает всех в Brawl Stars
- **"Плохая" (комедийная):** Все аниматроники орут "С 8 МАРТА" хором и пугают охранника
